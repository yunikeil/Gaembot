games.CirulliGame¶

class games.CirulliGame.GameCirulliView(size: int)¶

Класс для игры в 2048.

Атрибуты:

size (int): размер поля игры.

Методы:

drow_matrix: отрисовка матрицы игры. generate_points: генерация новой точки в матрице. move_up_button: кнопка для движения вверх. move_down_button: кнопка для движения вниз. move_left_button: кнопка для движения влево. move_right_button: кнопка для движения вправо.
